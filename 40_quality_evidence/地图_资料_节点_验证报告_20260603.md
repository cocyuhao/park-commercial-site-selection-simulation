## 已完成修改

| 目标 | 完成情况 |
|---|---|
| 真实高德 JS 地图 | 页面使用高德 JS API 2.0 渲染，检查到高德 canvas 和 AutoNavi 版权 |
| 不再用静态图假装地图 | 空间地图主路径不再回退展示静态图层 |
| 拖拽与缩放 | 拖拽、滚轮缩放交给高德地图；按钮缩放调用高德地图对象 |
| 容器变化 | 使用 `ResizeObserver`，变化后执行 `resize` 和 `setFitView` |
| 标记分层 | 当前项目位置、候选节点、商业 POI、搜索结果使用不同样式 |
| POI 信息卡 | 点击 POI 显示名称、类型、距离、来源、待复核状态 |
| 结果列表 | 右侧增加简洁 POI 结果列表，可点击定位并同步信息卡 |
| 搜索失败 | 统一显示“地图检索暂时失败，可以稍后重试，或先手动输入项目位置。” |
| 失败保护 | 搜索失败保留输入，地图不崩，不显示后端异常原文 |
