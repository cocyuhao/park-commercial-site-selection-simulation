# 老板方法资料归档说明

本目录用于归档和研究用户本轮说明的 6 份方法资料：1 张长图、2 个 DOCX、3 个 PDF。

使用边界：

- 这些资料优先作为“人物仿真 / 行为仿真 / 供需缺口 / AI Agent 系统设计”的方法资料和思路资料。
- 旧项目中的 PDF、CSV、证据台账等仍作为仿真数据来源；不能把方法资料中的示例数字直接当成 checked 证据。
- PPT 只学习汇报角度、专业语言、结构层级，不学习未核实的内容结论。
- 所有由 DeepSeek 或其他 LLM 生成的画像、概率、路线、消费逻辑，必须保持 `needs_review / not_final`，并保留可编辑、可删除、可替换的用户自主权。

后续产物：

- `source_manifest.csv`：本轮 6 份资料定位和复制清单。
- `extracted/`：文本、图片信息、页数、结构化摘要。
- `method_knowledge_base.md`：可交接方法知识库。
- `simulation_accuracy_plan.md`：提高人物仿真准确性的方案与落地路径。
